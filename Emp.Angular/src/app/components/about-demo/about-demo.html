Project Overview

  • Showcases skills and experience gained from previous industry projects.

  • On merge to the Main branch of Git, the CI pipeline initiates a build and runs all XUnit tests.

Backend Architecture

Emp.Api

  • Handles routing, authentication, and delegates logic to Application layer.
  • Includes Controllers, DI setup, Middleware, Filters, Serilog, appsettings.json.
  • References: Emp.Application, Emp.Core, Emp.Infrastructure

Emp.Application

  • Contains business logic and orchestrates operations using DTOs.
  • References: Emp.Core, Emp.Infrastructure

Emp.Core

  • Domain layer with entities, DTOs, enums, validation attributes, interfaces.
  • References: None

Emp.Infrastructure

  • Implements data access, migrations, repositories, Unit of Work.
  • References: Emp.Core, EF Core packages

Emp.Tests

  • xUnit test project for Controllers, Services, Repositories.
  • Uses xUnit, Moq, EF Core InMemory
